# Installation

Run the `apps.ps1` script to install the packages.

# Requirements

- `winget` support.

### Dev

- Windows Terminal
- Visual Studio Code
- Git
- Ubuntu 20.04

### Browser

- Mozilla Firefox

### Tools

- Power Toys

### Drivers

- GeForce Experience

### Comunication

- Discord

### Tools

- Authy
- Dropbox

# More

To include new packages, add them to the `apps.ps1` script.

To know the package name, visit [winget.run](https://winget.run/).
